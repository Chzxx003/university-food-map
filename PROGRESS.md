## 四、功能实现详情

### 4.1 P0 — 核心功能

#### 餐厅列表
- 卡片式展示：封面图、名称、菜系标签、人均价格、评分、距离、场景标签
- 支持**网格视图**与**列表视图**一键切换
- 列表根据筛选条件实时过滤，按距离或评分排序
- 图片懒加载（`loading="lazy"`）

#### 地图展示
- 使用 Leaflet + OpenStreetMap 瓦片，无需 API Key
- 红色 Pin 标注所有餐厅位置
- 点击 Pin 弹出餐厅简要信息卡片（含封面图、评分、价格）
- 切换学校时地图自动飞行到对应位置（`flyTo` 动画）
- 支持 PC 端地图与列表并排展示

#### 搜索与筛选
- **搜索**：支持按餐厅名称、菜系、菜品名称模糊搜索
- **菜系筛选**：川菜、日料、韩料、西餐、快餐、小吃、甜品、烧烤等 12 种
- **场景筛选**：一人食、聚餐、宵夜、约会、学生套餐、外卖友好、自习续杯
- **价格筛选**：15 元以下 / 15-30 元 / 30-50 元 / 50 元以上
- **评分筛选**：不限 / 4 星+ / 4.5 星+
- **营业状态**：仅看营业中
- 支持一键清除全部筛选

#### 餐厅详情页
- 图片轮播（左右箭头 + 底部指示器）
- 基本信息：名称、地址、电话、营业时间、人均价格、营业状态
- 场景标签展示
- 推荐菜品卡片（图片 + 名称 + 价格，2×2 网格）
- 迷你地图定位（Leaflet，禁止拖拽/缩放，纯展示）
- 用户评价列表 + 评价提交表单

### 4.2 P1 — 重要功能

#### 学生场景标签
- 每家餐厅标注 1~3 个场景标签
- 标签可用于筛选，也在卡片和详情页展示
- 7 种标签：一人食、聚餐、宵夜、约会、学生套餐、外卖友好、自习续杯

#### 排行榜
- **本周热门榜**：按访问量排序
- **高分好评榜**：按综合评分排序
- **高性价比榜**：按 评分/人均价格 比值排序
- 前三名展示特殊徽章（皇冠、银牌、铜牌）
- 点击可跳转详情页

#### 评价与评分
- 用户可提交：昵称（选填）+ 1-5 星评分 + 文字评价（10-200 字）
- 评价实时追加到列表顶部
- 评价数据通过 localStorage 持久化
- 支持点赞按钮（UI 已实现）

### 4.3 P2 — 加分功能

#### 今天吃什么（随机推荐）
- 点击导航栏「帮我选」按钮打开弹窗
- 随机滚动动画效果（100ms 间隔，12 次翻转）
- 展示推荐餐厅的封面、评分、价格、地址
- 「换一个」重新随机 / 「就这家」跳转详情页
- 基于当前筛选条件进行随机（不是全量随机）

#### 收藏夹
- 卡片右上角爱心按钮一键收藏/取消
- 详情页顶部也有收藏按钮
- 收藏数据存储在 localStorage，刷新不丢失
- 独立收藏页面展示所有已收藏餐厅
- 空状态友好提示

#### 按学校分区
- 首页顶部学校切换 Tab（全部 + 5 所学校）
- 切换后列表仅显示该校周边餐厅
- 地图自动飞行到对应学校位置
- 餐厅按距离排序（基于 Haversine 公式计算）
